Welcome to the Visio-Verbal Teleimpedance Backend

This server powers the visio-verbal teleimpedance interface by integrating voice, images, and gaze data.

For interactive API documentation and testing, please visit the /docs endpoint.

Available Endpoints

  • GET /reset - Resets the conversation history. 
    Example Response:
{{
"message": "Conversation history reset."
}}
  • POST /register_webhook - Registers a webhook URL. 
    Example Request Body (JSON):
{{
"webhook_url": "http://example.com/webhook"
}}
            
    Example Response:
{{
"message": "Webhook registered successfully."
}}
  • POST /unregister_webhook - Unregisters a webhook URL. 
    Example Request Body (JSON):
{{
"webhook_url": "http://example.com/webhook"
}}
            
    Example Response:
{{
"message": "Webhook unregistered successfully."
}}
  • GET /list_webhooks - Lists all registered webhook URLs. 
    Example Response:
{{
"registered_webhooks": [
"http://example.com/webhook1",
"http://example.com/webhook2"
]
}}
  • POST /upload_image - Uploads an image and processes it. 
    Example Request:
Multipart/form-data with:
- **file**: Image file to upload
            
    Example Response:
{{
"file_url": "http://backend.example.com/static/images/uploaded-image.jpg"
}}
  • POST /post_audio - Processes uploaded audio and generates a response. 
    Example Request:
Multipart/form-data with:
- **file**: Audio file to upload
- **image_url** (Optional): URL of an associated image
            
    Example Response:
(Streaming audio response)
Headers:
- **x-matrix-url**: URL to the stiffness matrix CSV file
- **x-ellipsoid-url**: URL to the ellipsoid plot image
  • GET /calibrate - Initiates calibration of the connected eye tracker. 
    Example Response:
{{
"status": "success",
"message": "Calibration complete"
}}
  • GET /capture_snapshot - Captures a snapshot with gaze point overlay. 
    Example Response:
{{
"status": "success",
"image_url": "http://backend.example.com/static/images/snapshot.jpg"
}}

Usage Instructions

Use these endpoints to interact with the visio-verbal teleimpedance system. This server integrates data from voice, image, and gaze trackers to process user inputs.

For interactive API documentation and to try out the endpoints directly from your browser, please visit the /docs endpoint.
